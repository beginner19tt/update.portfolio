## 🚀 Projects

### 📍 住所マッパー（チラシ配り効率化ツール）
アルバイト先での「数百件のチラシ配布ルートを手作業で整理する」という非効率な業務を解決するために自発的に開発したツールです。
* **概要:** PDFから住所データを読み取り、町名ごとにグループ化して最短ルートをナビアプリ（Google Maps, Yahoo!カーナビ, Apple Maps）へ出力します。
* **技術スタック:** JavaScript, HTML/CSS, PDF.js, QRCode.js, 各種Map API
* **成果:** AIを活用して要件定義から実装までを単独で完遂し、現場の業務時間を大幅に削減しました。

### 👁️ 画像認識を用いた異常検知システム
チーム開発（2年生）においてリーダーとして進行・開発を担当したプロジェクトです。
* **概要:** ResNet50やVGG16などの画像認識モデルを用いて、異常を自動で識別・判別するAIシステム。
* **技術スタック:** Python, Machine Learning (ResNet50, VGG16)
* **成果:** チームの成果を最大化するため「1人1モデルを作成し最高精度を提出する」戦略を採用し、全体で3位の成績を獲得。同時に、進捗管理やタスク分散といったマネジメントの重要性を学びました。

### 🍻 幹事代行ChatBot
大人数のアサインや日程・お店調整の負担を減らすことをターゲットにしたLINE Botです。
* **概要:** LINE上で人数や場所、予算などの条件を入力すると、現在地から条件に合う飲食店を提案します。
* **技術スタック:** Python, AWS, LINE Messaging API, Gemini API, ホットペッパーAPI
* **成果:** 複数の外部APIを連携させる実践的な開発を経験し、APIの仕様変更への対応や例外処理（Webにない店舗情報の処理など）の課題解決に取り組みました。

### 💰 バイトモチベアプリ
学生向けの給与・貯金管理Webアプリケーションです。（現在開発中）
* **概要:** アルバイトのモチベーション維持や目標管理を目的としたシステム。
* **技術スタック:** Python, Flask, SQL
