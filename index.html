自己紹介

こんにちは！福岡の工業高校でC言語を3年間学んだ後、麻生情報ビジネス専門学校でAI・IoTを専攻しています。

私の強みは「身近な課題を発見し、技術と実行力で解決に導く力」です。アルバイト先の手作業ルート整理を自発的にツール化し、平均5分の作業を10秒に短縮した経験がその象徴です。失敗を客観的に分析して仕組みで改善するINTP-Aらしいアプローチを大切にしています。

将来的にAIが実装の多くを代替する時代を見据え、要件定義から実装・運用まで上流から下流まで一貫して携われる環境で技術力を磨きたいと考えています。最終的には、顧客の潜在的な課題を引き出しチームを牽引するプロジェクトマネージャー（PM）を目指しています。

住所マッパー（チラシ配り効率化ツール）

アルバイト先で200〜300件のチラシ配布ルートを手作業で整理する非効率な業務を発見。PDFから住所を読み取り、町名ごとにグループ化して最短ルートをGoogle Maps・Yahoo!カーナビ・Apple Mapsへ出力するツールを自発的に開発。AIを活用して設計から実装まで単独でやり切った。

⚡ 平均5分かかっていた作業を10秒に短縮
JavaScript PDF.js QRCode.js Map API
チーム開発・リーダー

画像認識を用いた不良品検知システム

2年生のチーム開発でリーダーを担当。ResNet50・VGG16を活用した異常検知AIを構築。「1人1モデルを作成し最高精度を提出する」という戦略でチームの成果を最大化し全体3位を獲得。進捗確認のルール化やタスクの適切な分散など、マネジメントの実践的な学びを得た。

🏆 コンペ全体3位獲得
